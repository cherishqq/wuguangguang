Add extra meta fields for user account owners to provide inputs in profiles. Easy to use, simple and dynamic configurations.

== Description ==

This plugin provides a configuration form in the WordPress admin dashboard to enable custom user profile form fields.
This version of the plugin is fully functional to get started with managing meta details on your user accounts.
If you find this plugin useful, please consider a small donation to support the development of this kind of work.
For a small donation, several extra features have be enabled in the plugin that are extremely useful.
<br>
Please visit the URL:
<a href="http://www.feweb.net/wordpressplugin_extrauserfields.htm">http://www.feweb.net/wordpressplugin_extrauserfields.htm</a>
for details and screenshots on extra features such as:
<ul>
<li>Admin-only control fields in addition to fields managable by account owners. (Separate custom fields that are read-only to the member account)</li>
<li>Datepicker integration as a mask on input text fields.</li>
<li>Input mask for text inputs like telephone numbers and credit cards.</li>
<li>Predefined form values for US States and Countries</li>
<li>Extended search features on the "All Users" page using the custom user meta fields defined.</li>
<li>Customizable display of user meta fields in the "All Users" page.</li>
<li>Export feature to download user data to Tab separated values file for use with spreadsheet workbooks.</li>
<li>Shortcode support to display user meta information in posts and pages.</li>
<li>Extended field customizations to temporarily enable/disable saved fields.</li>
</ul>

== Features ==

* Easy to configure form fields of types TEXT, CHECKBOX, RADIO, SELECT and TEXTAREA with singular and multiple value options.
* Displays custom fields on the user profile page for management by the admin account and the account owner.
* CSS ready for 100% customization
* Full system logging via syslog

== Installation ==

1. Dowload the plugin to your WP site.
2. Enabled the plugin.
3. Define custom form fields using the simple syntax defined to enable various profile form fields.
4. See the menu: Settings >> Extra User Fields

== Frequently Asked Questions ==

= Can I provide multiple values to my form fields? =

Yes. The SELECT, RADIO and even the CHECKBOX custom fields allow for multiple values.

= Can I customize the look and feel of the forms? =

Yes. The HTML in the forms is designed for easy CSS selection and isolation of styling as needed. All the forms pull in the active theme header and footer.
